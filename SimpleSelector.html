Simple Selectors

  1. What is Element Selector ?
    • Element selector select the html element on the basis of the name.
  2. What is CSS id selector?
    • Id selector uses the id attribute of an html element to specific element.
    • Id is unique,so id selector is used to select one unique element
    • To select an element with an specific id ,write a hash character(#)followed by the id of the element
    • An id name cannot start with a number!
  3. What is CSS Class Selector?
    • CSS class selector selects HTML Element with specific class attribute
    • To select element with specific class start with period(.)& followed by class name
